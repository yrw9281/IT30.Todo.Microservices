# Day 01 - 前言與大綱

## 前言

在這個時代，大型網站設計正朝著微服務架構的方向快速發展。微服務不僅改變了我們構建和部署應用的方式，還賦予了開發者更多的靈活性與可擴展性。隨著微服務開發逐漸成為主流，許多新的設計框架與概念不斷推陳出新。本系列旨在帶領讀者深入了解現代微服務的宏觀架構，並引導你設計與實作一個微服務系統。

## 閱讀前的背景知識

1. .NET C# 的基礎知識
2. Docker 容器化的基礎知識
